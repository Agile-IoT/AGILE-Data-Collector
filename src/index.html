This is a basic implementation. You introduce a url, then click "Fetch Data" in order to load some random placeholder data. Once that is done, you can introduce the uri of a certain Solid server for storage, and click "Upload Data", if everything went right, you will be presented with a window asking you to choose a certificate in order to authenticate yourself with the Solid server. The upload will succeed only if you select a valid certificate, and the Solid server allows the selected user to upload data to the particular directory. You can also press "Read" to read whatever data is at the introduced Solid url.
